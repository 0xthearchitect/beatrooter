## São tão lindos quando são bébés!

Criamos em python uma pequena agente chamada Nábia, com regressão logistica e um pequeno modelo de dados à mão, tendo como dados um número limitado de vulnerabilidades!
Este foi o começo para a Nábia, que vinha a ser uma das maiores agentes de IA na cibersegurança. O sonho dela é conhecer o ídolo dela quando crescer, o Sr. Sonnet.
